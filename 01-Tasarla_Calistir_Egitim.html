🔀 Koşul Yapıları (if / else if / else)

Robot karar verecekse if kullanıyoruz. Engel var mı? Çizgide mi? Bunların hepsine koşulla karar veriyoruz.

kosullar.ino
int mesafe = 30; // cm cinsinden

// Temel if - else
if (mesafe < 20) {
  Serial.println("ENGEL VAR! Dur.");
} else {
  Serial.println("Yol açık, devam.");
}

// Çoklu koşul: if / else if / else
if (mesafe < 10) {
  Serial.println("Çok yakın! Geri dön.");
} else if (mesafe < 25) {
  Serial.println("Dikkatli ilerle.");
} else {
  Serial.println("Hızlı gidebilirsin.");
}

// Mantıksal operatörler
bool sol_engel = true;
bool sag_engel = false;

if (sol_engel && sag_engel) {       // her ikisi de true?
  Serial.println("Her iki yanda engel!");
}
if (sol_engel || sag_engel) {       // en az biri true?
  Serial.println("En az bir yanda engel.");
}
if (!sag_engel) {                    // sag_engel false ise true
  Serial.println("Sağ yol açık.");
}

🔁 Döngü Yapıları (for / while)

Aynı şeyi defalarca yapmak istiyorsan döngü kullanırsın. Yeterlilik sınavında for döngüsü ile pin tanımlama sorusu kesinlikle çıkıyor!

donguler.ino
// ── FOR DÖNGÜSÜ ──────────────────────────────────────
// for(başlangıç; koşul; artış)

// 0'dan 9'a kadar say (10 kez)
for (int i = 0; i < 10; i++) {
  Serial.println(i);
}

// ★ SINAV SORUSU: 3'ten 9'a kadar (dahil) pinleri OUTPUT yap
for (int i = 3; i <= 9; i++) {
  pinMode(i, OUTPUT);
}

// ── WHILE DÖNGÜSÜ ────────────────────────────────────
// Koşul doğru olduğu sürece çalışır
int sayac = 0;
while (sayac < 5) {
  Serial.println(sayac);
  sayac++;   // sayac = sayac + 1
}

// loop() zaten sonsuz döngüdür — ek while'a genelde gerek yok!
⚠️
Sınav not: Sorusu şöyle gelir: "3 ile 9. pinleri dahil OUTPUT yapan for döngüsü hangisi?" Cevap: for(int i=3; i<=9; i++) { pinMode(i,OUTPUT); } — büyük/küçük harf önemli! OUTPUT büyük, output değil.

Bölüm 02

Arduino Temel Fonksiyonları

Arduino'da sıkça kullandığın hazır komutlar. Bunları ezberlemene gerek yok — anlarsan hatırlarsın.

⚙️

pinMode(pin, mod)

Bir pini giriş mi yoksa çıkış mı olarak kullanacağını belirtir. setup() içinde yapılır.

OUTPUT INPUT INPUT_PULLUP
💡

digitalWrite(pin, değer)

Dijital çıkış pinini aç ya da kapat. LED yakmak, motor yönü vermek için kullanılır.

HIGH = 5VLOW = 0V
📥

digitalRead(pin)

Dijital giriş pininin durumunu okur. Düğme basılı mı? Çizgi sensöründen 0/1 sinyali geliyor mu?

HIGH = 1LOW = 0
📊

analogRead(pin)

Analog giriş pininden 0–1023 arası değer okur. Işık sensörü, 8'li çizgi sensörü için kullanılır. Sadece A0–A5 pinlerinde çalışır.

0 … 1023A0–A5
🎛️

analogWrite(pin, değer)

PWM sinyali üretir. Motor hızını ayarlamak için kullanılır. 0 = dur, 255 = tam hız. Sadece ~ (tilde) işaretli pinlerde çalışır.

0 … 255PWM pinleri
⏱️

delay(ms) ve millis()

delay(1000) = 1 saniye bekle. millis() = başlangıçtan kaç milisaniye geçti? Robotun takılmaması için millis() tercih edilir.

1000ms = 1s
📏

pulseIn(pin, tip)

Ultrasonik sensörden gelen darbe süresini ölçer. HC-SR04 ile mesafe hesaplamada kullanılır. Mikrosaniye (µs) döndürür.

HIGH / LOWµs döner
🖥️

Serial.begin / println

Bilgisayar ile iletişim. Serial.begin(9600) bağlantıyı açar, Serial.println(değer) ekrana yazar. Test sırasında çok işe yarar.

9600 baud
temel_fonksiyonlar_ozet.ino
// Tüm temel fonksiyonlar bir arada

const int LED_PIN = 13;
const int DUGME_PIN = 2;
const int SENSOR_PIN = A0;
const int MOTOR_HIZ = 9;  // ~ işaretli pin

void setup() {
  Serial.begin(9600);           // Seri haberleşme başlat
  pinMode(LED_PIN, OUTPUT);    // LED: çıkış
  pinMode(DUGME_PIN, INPUT);   // Düğme: giriş
}

void loop() {
  // Dijital okuma
  int dugme = digitalRead(DUGME_PIN);
  
  // Analog okuma
  int sensorDegeri = analogRead(SENSOR_PIN);  // 0-1023
  
  // Koşullu LED kontrolü
  if (dugme == HIGH) {
    digitalWrite(LED_PIN, HIGH);  // LED yak
  } else {
    digitalWrite(LED_PIN, LOW);   // LED söndür
  }
  
  // Motor hız kontrolü (PWM)
  analogWrite(MOTOR_HIZ, 200);   // 0-255 arası hız
  
  // Seri monitöre yazdır
  Serial.print("Sensor: ");
  Serial.println(sensorDegeri);
  
  delay(100);
}
Bölüm 03

Dijital & Analog Giriş/Çıkış

Sensörler nasıl konuşur? Motorlara nasıl komut verilir? Burada öğreniyoruz.

⬛ Dijital — 0 ya da 1

Sadece iki durum var: HIGH (5V) veya LOW (0V). Işık ya açık ya kapalı gibi düşün.

  • LED yakma/söndürme
  • Çizgi sensöründen siyah/beyaz okuma
  • Motor yön kontrolü (İleri/Geri)
  • Düğme okuma

📈 Analog — 0'dan 1023'e

Sonsuz değer alabilen sinyal. Sesteki perde gibi — alçaktan yükseğe kesintisiz değişir.

  • 8'li çizgi sensöründen hassas okuma
  • Ultrasonik mesafe hesaplama
  • Motor hız kontrolü (0–255 PWM)
  • Işık sensörü, potansiyometre

📡 HC-SR04 Ultrasonik Mesafe Sensörü

Ses dalgasının gidip gelme süresini ölçerek mesafeyi hesaplar. 2–450 cm arası çalışır. Yarışmada engel algılama için kullanırsın.

PinGörevBağlantı
VCCGüç5V'a bağla
GNDToprakGND'a bağla
TRIGTetikHerhangi dijital pin (ör. 7)
ECHOYankıHerhangi dijital pin (ör. 8)
📐
Mesafe formülü: Ses 340 m/s = 0.034 cm/µs hızla gider. Gidip gelir = 2 kat mesafe. Bu yüzden sure / 58 veya sure * 0.034 / 2
hcsr04.ino
const int TRIG = 7;
const int ECHO = 8;

void setup() {
  Serial.begin(9600);
  pinMode(TRIG, OUTPUT);
  pinMode(ECHO, INPUT);
}

long mesafeOku() {
  digitalWrite(TRIG, LOW);
  delayMicroseconds(2);
  digitalWrite(TRIG, HIGH);
  delayMicroseconds(10);
  digitalWrite(TRIG, LOW);
  long sure = pulseIn(ECHO, HIGH);
  return sure / 58; // cm
}

void loop() {
  long cm = mesafeOku();
  Serial.print("Mesafe: ");
  Serial.print(cm);
  Serial.println(" cm");
  delay(200);
}

⚙️ L298N Motor Sürücü Shield & DC Motorlar

Arduino tek başına motoru döndürecek güce sahip değil. L298N, küçük Arduino sinyalini güçlü motor sinyaline çevirir. Shield (kalkan) versiyonu doğrudan Arduino'nun üstüne takılır.

motor_surucü.ino
// L298N Shield - Motor A: pin 3,4,5 / Motor B: pin 6,7,8 (örnek)
// Gerçek pin numaraları kullandığın shield'e göre değişir!

// Motor A pinleri
const int MA_HIZ = 5;   // PWM ~ pini
const int MA_IN1 = 3;
const int MA_IN2 = 4;

// Motor B pinleri
const int MB_HIZ = 6;   // PWM ~ pini
const int MB_IN1 = 7;
const int MB_IN2 = 8;

void setup() {
  pinMode(MA_HIZ,OUTPUT); pinMode(MA_IN1,OUTPUT); pinMode(MA_IN2,OUTPUT);
  pinMode(MB_HIZ,OUTPUT); pinMode(MB_IN1,OUTPUT); pinMode(MB_IN2,OUTPUT);
}

void ileri(int hiz) {
  analogWrite(MA_HIZ, hiz); digitalWrite(MA_IN1, HIGH); digitalWrite(MA_IN2, LOW);
  analogWrite(MB_HIZ, hiz); digitalWrite(MB_IN1, HIGH); digitalWrite(MB_IN2, LOW);
}

void geri(int hiz) {
  analogWrite(MA_HIZ, hiz); digitalWrite(MA_IN1, LOW); digitalWrite(MA_IN2, HIGH);
  analogWrite(MB_HIZ, hiz); digitalWrite(MB_IN1, LOW); digitalWrite(MB_IN2, HIGH);
}

void sagaDon(int hiz) {  // Sol motor hızlı, sağ yavaş
  analogWrite(MA_HIZ, hiz); digitalWrite(MA_IN1, HIGH); digitalWrite(MA_IN2, LOW);
  analogWrite(MB_HIZ, 0);   digitalWrite(MB_IN1, LOW);  digitalWrite(MB_IN2, LOW);
}

void solaDon(int hiz) {   // Sağ motor hızlı, sol yavaş
  analogWrite(MA_HIZ, 0);   digitalWrite(MA_IN1, LOW);  digitalWrite(MA_IN2, LOW);
  analogWrite(MB_HIZ, hiz); digitalWrite(MB_IN1, HIGH); digitalWrite(MB_IN2, LOW);
}

void dur() {
  analogWrite(MA_HIZ, 0); analogWrite(MB_HIZ, 0);
}

void loop() {
  ileri(200);   delay(2000);
  sagaDon(180); delay(500);
  dur();        delay(1000);
}
💡
İpucu: Her motorun yönü IN1/IN2 ile belirlenir: IN1=HIGH, IN2=LOW → ileri. Tersi → geri. Hız ise ENA/ENB ya da PWM hız pinine analogWrite() ile 0–255 arası değer gönderilerek ayarlanır.

🖤 Çizgi Sensörü Kullanımı

Yarışmada iki farklı çizgi sensörün var: 8'li analog sensör ve 4'lü dijital sensör. İkisi de IR ışınıyla çalışır — siyah çizgiyi beyaz zeminden ayırt eder.

8'li Analog Sensör

Her sensörden 0–1023 arası değer okursun. Hangi sensörün çizgide olduğunu ve ne kadar sapma olduğunu anlarsın. PID için idealdir.

  • Siyah çizgi üzerinde: düşük değer (~100)
  • Beyaz zemin üzerinde: yüksek değer (~900)
  • Eşik değeri genelde 500 civarı

4'lü Dijital Sensör

Her sensörden sadece 0 ya da 1 okursun. Basit çizgi izleme için yeterli — çok hızlı ve kolay kullanılır.

  • Siyah çizgi: 0 (LOW)
  • Beyaz zemin: 1 (HIGH)
  • Sol/Sağ/Orta karşılaştırması yapılır
cizgi_izleme_temel.ino
// 4'lü dijital çizgi sensörü ile basit çizgi izleme
// Sensör pinleri (sol → sağ)
const int S_SOL  = 10;
const int S_MSOL = 11;
const int S_MSAG = 12;
const int S_SAG  = 13;

void setup() {
  Serial.begin(9600);
  // Motor pinleri de tanımlanmalı...
}

void loop() {
  int sl = digitalRead(S_SOL);
  int sml = digitalRead(S_MSOL);
  int smr = digitalRead(S_MSAG);
  int sr = digitalRead(S_SAG);

  // Düz çizgi: ortadaki ikisi çizgide
  if (sml == 0 && smr == 0) {
    ileri(180);
  }
  // Sola kaydı: sol sensörler çizgide
  else if (sl == 0 || sml == 0) {
    solaDon(150);
  }
  // Sağa kaydı: sağ sensörler çizgide
  else if (sr == 0 || smr == 0) {
    sagaDon(150);
  }
  // Hiçbiri çizgide değil: dur
  else {
    dur();
  }
}

⚡ Ohm Yasası: V = I × R

Üç değerden ikisini bilirsen üçüncüyü bulabilirsin:

🔗 Seri ve Paralel Devre

Seri Devre

Elemanlar arka arkaya bağlıdır. Tüm elemandan aynı akım geçer.

  • Direnç: R_toplam = R1 + R2 + ...
  • Bir lamba söner → hepsi söner
  • Her lambada voltaj azalır
  • Parlak mı? Seri devrede lambalar SÖNÜK yanar

Paralel Devre

Elemanlar yan yana bağlıdır. Her elemanda aynı voltaj bulunur.

  • Direnç: 1/R = 1/R1 + 1/R2 + ...
  • Bir lamba söner → diğerleri yanmaya devam
  • Her lambada tam voltaj var
  • Parlak mı? Paralel devrede lambalar PARLAK yanar
🎯
Sınav sorusu: "Seri bağlı lambalar mı parlak, paralel bağlı mı?" → Paralel bağlı lambalar parlaktır çünkü her biri tam voltajı alır. Seri bağlıda voltaj paylaşıldığından lambalar daha sönük yanar.

📋 Basit Elektrik Devresinin Zorunlu Elemanları

Yarışma sorusunda "basit elektrik devresinde hangisi bulunmaz?" diye soruluyor.

✅ Zorunlu Elemanlar

  • Pil (enerji kaynağı)
  • İletken kablo (akım yolu)
  • Ampul / tüketici (enerjiyi kullanan)

❌ Zorunlu Değil (Ama Kullanılabilir)

  • Sigorta (koruyucu, şart değil)
  • Anahtar (kontrol, şart değil)
  • Voltmetre / Ampermetre (ölçüm)

🤖 Temel Engelden Kaçan Robot Kodu (Başlangıç Noktası)

Bu kodu doğrudan kullanabilirsin ama pin numaralarını kendi bağlantına göre değiştirmeyi unutma!

engel_kacma_robot.ino
// ────────────────────────────────────────────────────
// Temel Engelden Kaçan Robot
// HC-SR04 ultrasonik sensör + L298N motor sürücü shield
// ────────────────────────────────────────────────────

// Motor A (Sol)
const int ENA = 5;  // PWM hız
const int IN1 = 3;
const int IN2 = 4;

// Ön ultrasonik sensör
const int TRIG = 9;
const int ECHO = 10;

const int ENGEL_CM = 25;  // kaç cm'de dur?
const int MOTOR_HIZ = 180; // 0-255

void setup() {
  Serial.begin(9600);
  pinMode(ENA,OUTPUT); pinMode(IN1,OUTPUT); pinMode(IN2,OUTPUT);
  pinMode(ENB,OUTPUT); pinMode(IN3,OUTPUT); pinMode(IN4,OUTPUT);
  pinMode(TRIG,OUTPUT); pinMode(ECHO,INPUT);
  Serial.println("Robot hazır!");
}

long mesafeOku() {
  digitalWrite(TRIG, LOW);  delayMicroseconds(2);
  digitalWrite(TRIG, HIGH); delayMicroseconds(10);
  digitalWrite(TRIG, LOW);
  long sure = pulseIn(ECHO, HIGH);
  return sure / 58;
}

void ileri() {
  analogWrite(ENA, MOTOR_HIZ); digitalWrite(IN1,HIGH); digitalWrite(IN2,LOW);
  analogWrite(ENB, MOTOR_HIZ); digitalWrite(IN3,HIGH); digitalWrite(IN4,LOW);
}
void geri() {
  analogWrite(ENA, MOTOR_HIZ); digitalWrite(IN1,LOW); digitalWrite(IN2,HIGH);
  analogWrite(ENB, MOTOR_HIZ); digitalWrite(IN3,LOW); digitalWrite(IN4,HIGH);
}
void sagaDon() {
  analogWrite(ENA, MOTOR_HIZ); digitalWrite(IN1,HIGH); digitalWrite(IN2,LOW);
  analogWrite(ENB, 0);         digitalWrite(IN3,LOW);  digitalWrite(IN4,LOW);
}
void dur() {
  analogWrite(ENA,0); analogWrite(ENB,0);
}

void loop() {
  long cm = mesafeOku();
  Serial.print("Mesafe: "); Serial.print(cm); Serial.println(" cm");

  if (cm < ENGEL_CM && cm > 0) {
    dur();          delay(300);
    geri();         delay(500);
    sagaDon();      delay(600);
  } else {
    ileri();
  }
  delay(50);
}

// Çizgi İzleyen Robot — 4'lü dijital sensör versiyonu
// 0 = siyah çizgi üzerinde, 1 = beyaz zemin

// Sensör pinleri
const int SOL_D  = 10;  // en sol
const int SOL_O  = 11;  // sol orta
const int SAG_O  = 12;  // sağ orta
const int SAG_D  = 13;  // en sağ

// Motor pinleri
const int ENA = 5, IN1 = 3, IN2 = 4;  // Sol motor
const int ENB = 6, IN3 = 7, IN4 = 8;  // Sağ motor

const int HIZ_NOR = 180;
const int HIZ_DON = 150;

void setup() {
  for (int i = 3; i <= 8; i++) pinMode(i, OUTPUT);  // motor pinleri
  pinMode(SOL_D,INPUT); pinMode(SOL_O,INPUT);
  pinMode(SAG_O,INPUT); pinMode(SAG_D,INPUT);
}

void motorSol(int hiz, bool ileri) {
  analogWrite(ENA, hiz);
  digitalWrite(IN1, ileri ? HIGH : LOW);
  digitalWrite(IN2, ileri ? LOW : HIGH);
}
void motorSag(int hiz, bool ileri) {
  analogWrite(ENB, hiz);
  digitalWrite(IN3, ileri ? HIGH : LOW);
  digitalWrite(IN4, ileri ? LOW : HIGH);
}

void loop() {
  int sl = digitalRead(SOL_D);
  int so = digitalRead(SOL_O);
  int ro = digitalRead(SAG_O);
  int sr = digitalRead(SAG_D);

  if (so == 0 && ro == 0) {       // ✅ Düz git
    motorSol(HIZ_NOR, true);
    motorSag(HIZ_NOR, true);
  } else if (sl == 0 || so == 0) { // ↩ Sola dön
    motorSol(0, true);
    motorSag(HIZ_DON, true);
  } else if (sr == 0 || ro == 0) { // ↪ Sağa dön
    motorSol(HIZ_DON, true);
    motorSag(0, true);
  } else {                           // Çizgi yok: dur
    motorSol(0, true);
    motorSag(0, true);
  }
}
